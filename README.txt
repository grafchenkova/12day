QUANTUM START - LANDING PAGE
=============================

Структура файлов:
- index.html        : Главная HTML страница (содержит CSS и JS)
- style.css         : Стили (уже встроены в index.html)
- assets/           : Папка с ресурсами
  - images/         : Изображения (фон, текстуры, портрет автора)
  - index-*.css     : Скомпилированные стили
  - index-*.js      : Скомпилированный JavaScript

ИСПОЛЬЗОВАНИЕ:
1. Откройте index.html в браузере
2. Все стили и скрипты уже встроены в файл
3. Убедитесь, что папка assets находится в одной директории с index.html

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
- Фреймворк: React 19 + Tailwind CSS 4
- Дизайн: Brutalist Luxury (чёрный минимализм + золото)
- Шрифты: Cinzel (заголовки), Space Mono (текст)
- Интерактивность: Таймер обратного отсчёта, аккордеоны, плавные анимации

ПОДДЕРЖИВАЕМЫЕ БРАУЗЕРЫ:
- Chrome/Edge (последние версии)
- Firefox (последние версии)
- Safari (последние версии)
